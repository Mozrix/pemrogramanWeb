HTML Dasar

H1 pada html itu untuk menunjukkan heading utama dalam dokumen HTML.

  1. HTML adalah singkatan dari HyperText Markup Language.
  2. HTML digunakan untuk membuat halaman web.
  3. HTML terdiri dari elemen-elemen yang membentuk struktur halaman web.
  • HTML menggunakan tag untuk menandai elemen-elemen dalam halaman web.
  • HTML dapat digunakan bersama dengan CSS dan JavaScript untuk membuat halaman web yang lebih interaktif dan menarik.
